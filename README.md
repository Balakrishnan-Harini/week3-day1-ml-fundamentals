# Week 3 Day 1 - ML Fundamentals

## Project Overview

This project demonstrates the basics of Machine Learning using a Student Performance Dataset. The project includes dataset exploration, feature selection, train-test split, model training using Linear Regression, predictions, model evaluation, and visualizations.

## Dataset

The dataset contains:

* Study Hours
* Attendance
* Assignments
* Marks

## Tasks Completed

* Dataset Exploration
* Feature and Label Identification
* Machine Learning Workflow Documentation
* Train-Test Split
* Linear Regression Model Training
* Predictions
* Actual vs Predicted Comparison
* Model Evaluation
* Plotly Visualizations

## Technologies Used

* Python
* Pandas
* Scikit-Learn
* Plotly
* Jupyter Notebook

## How to Run

1. Open Jupyter Notebook
2. Open `assignment.ipynb`
3. Run all cells step by step

## Output

The project predicts student marks based on study hours, attendance, and assignments.
