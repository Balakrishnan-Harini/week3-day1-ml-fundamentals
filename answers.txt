1. What is slope?
   Slope is the coefficient value of each feature in Linear Regression. It shows how much the marks change when one feature increases by one unit.

2. What is intercept?
   Intercept is the starting value of prediction when all feature values become zero.

3. What do they mean?
   Slope tells the relationship between input features and output. Intercept gives the base prediction value of the model.

4. What is MAE?
   Mean Absolute Error measures the average difference between actual and predicted values. Lower MAE means better prediction.

5. What is R² Score?
   R² Score measures how well the model explains the data. A value close to 1 means better performance.

6. Is your model performing well?
   Yes, if the MAE is low and R² Score is close to 1, the model is performing well.
